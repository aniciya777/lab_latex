\begin{frame}{Выбор пакета для оформления программного кода}
    \hypersetup{pdfpagetransition={Box /D 2 /M /O}}
    
    Среди наиболее популярных, можно выделить следующие пакеты:
    
    \begin{itemize}
        \item \textbf{verbatim} --- вставка исходного кода в документ, без подсветки синтаксиса;
        \item \textbf{algorithm2e} --- для оформления псевдокодов и алгоритмов;
        \item \textbf{mcode} --- для подсветки синтаксиса и вставки программного кода \textit{MATLAB};
        \item \textbf{listings};
        \item \textbf{minted}.
    \end{itemize}
    
\end{frame}
